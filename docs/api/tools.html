tools package#

The package contains various wrappers of Omniverse tools and tools specific to Metron AI ArdaGen.

Submodules#

tools.isaac_sim

Implements Isaac App wrapper class.

tools.replicator

Implements Omniverse Replicator handler class.

tools.scenario

Implements Scenario class.

tools.scenarios_manager

Defines Scenarios Manager class which iterates over all scenarios.

tools.single_camera

Defines Single Camera class.

tools.writer

Defines OVWriter class to write generated data on a file system.
